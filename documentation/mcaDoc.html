mca Documentation

Mark Rivers

University of Chicago

mca supports Multi-Channel Analyzers (hence the name), and also related devices such as multi-channel scalers (MCS), waveform digitizers, and programmable ADCs, amplifiers and power supplies.

mca consists of:

  • The mcaRecord
  • Device support for the Canberra Ethernet MCAs and programmable instrumentation
  • Device support for the SIS 3801 MCS (formerly the STR7201 from Struck)
  • General device support for the MCA record using MPF servers.  This support is used by the DXP module, by the ip330 module, and by the quadEM module
  • C++ base class support for fastSweep devices.  This is used by the ip330 module and the quadEM module.
